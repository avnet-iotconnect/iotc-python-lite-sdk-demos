# Machine Learning Classifier Waveform Images

- `classifier_class0.svg`: triangle
- `classifier_class1.svg`: triangle mix
- `classifier_class2.svg`: burst + triangle
- `classifier_class3.svg` to `classifier_class5.svg`: parser accepts these values but Machine Learning Classifier behavior maps to class 2
