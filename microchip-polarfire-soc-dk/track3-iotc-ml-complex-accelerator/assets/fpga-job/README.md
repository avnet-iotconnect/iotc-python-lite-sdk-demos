# FPGA Job Asset

Track 3 FPGA programming artifact:

- `MPFS_DISCOVERY_KIT.job`

Quickstart flow programs this file with FlashPro Express before running the workshop package.

Implementation reports are staged in:

- `reports/MPFS_DISCOVERY_KIT_compile_netlist_resources.rpt` (utilization summary)
- `reports/MPFS_DISCOVERY_KIT_compile_netlist_hier_resources.csv` (hierarchical resource usage)
- `reports/MPFS_DISCOVERY_KIT_iteration_summary.rpt` (P&R iteration/timing summary)
- `reports/MPFS_DISCOVERY_KIT_timing_r3_s16.rpt` (timing analysis report)
- `reports/MPFS_DISCOVERY_KIT_timing_violations_max_r3_s16.rpt`
- `reports/MPFS_DISCOVERY_KIT_timing_violations_min_r3_s16.rpt`
- `reports/place_and_route_jitter_report.txt`
