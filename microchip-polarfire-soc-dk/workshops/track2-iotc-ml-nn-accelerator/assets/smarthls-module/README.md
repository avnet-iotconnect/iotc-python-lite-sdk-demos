# SmartHLS Module Payload

This directory contains the workshop `tinyml_nn` SmartHLS module source that is copied into the Microchip reference design tree.

Target location in reference design:

- `script_support/additional_configurations/smarthls/tinyml_nn/`

Use either:

- direct `Copy-Item` from this repository, or
- `tools/fetch-tinyml-nn-module.ps1` to pull from GitHub zip.
