# Track 2 Waveform Images

These SVG files show the Track-2 base signal equations (noise omitted).

- `track2_class0.svg`: triangle
- `track2_class1.svg`: triangle mix
- `track2_class2.svg`: burst + triangle
- `track2_class3.svg`: square
- `track2_class4.svg`: chirp-like sweep
- `track2_class5.svg`: impulse train
