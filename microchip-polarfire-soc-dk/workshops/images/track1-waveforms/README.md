# Track 1 Waveform Images

These SVG files show the Track-1 base signal equations (noise omitted).

- `track1_class0.svg`: triangle
- `track1_class1.svg`: triangle mix
- `track1_class2.svg`: burst + triangle
- `track1_class3.svg` to `track1_class5.svg`: parser accepts these values but Track-1 behavior maps to class 2
